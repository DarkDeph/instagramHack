pkg update && pkg upgrade - установка или обновка установленных пакетов.

pkg install [пакет] - установка нужного пакета. 

pip install [пакЭт] - ну я хз, пип инстал доступен.

termux-setup-storage - открытие проводника для термукса

ls - список файлов в нужной нам директории

cd - перемещение по директории

cd #home - автопилот, когда бухой лазяешь по директории и заблудился, тогда сразу попадаешь в начальную директорию. 

cd ..  - тут не использовалась но раз уж ты дочитал до этого то значит ты заинтересован, это вернутся назад в предыдущую директорию.

python [программа] - это запуск программы при помощи питона.
